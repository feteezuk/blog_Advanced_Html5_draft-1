Optimizing Images for the Web

Software/tool you used to optimize the image from the first page

Here I used the JPEG Optimizer to Resize and Compress my Photo that I have on my homepage.

what was the previous size of the file and what was your reduction

The original width was 960 pixels and the original height was 1280 pixels and the original filesize was 249.2 kb. I changed the width to become 300 pixels and the new height was automatically created to be 400 pixels. The new filesize was 14.7kb which gives me 94.1% savings in filesize.
